# GTNH-Japanese-Translation

## GT New Horizonsの日本語化パック

2023-12-04

初版公開

## 使い方

1、Releasesから対応するバージョンを探す

2、ダウンロード・解凍し、.minecraftフォルダに展開する

## クレジット

このプロジェクトは[GTNHの中国語翻訳チーム](https://github.com/Kiwi233/Translation-of-GTNH)からスクリプトをお借りしています。
